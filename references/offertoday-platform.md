# OfferToday 平台资料

> 2026-06-02 首次使用记录。

## 基本信息

- 正确网址：www.offertoday.com（不是 .com.hk）
- 技术架构：SPA（Vue/React），类似 BOSS直聘，API 路径 /wapi/*
- 有移动端 APP（Google Play: com.offerToday）
- API SDK 引用 static.zhipin.com（与 BOSS直聘共用基础设施）

## 页面路由

| 路由 | 功能 |
|------|------|
| /signup | 注册/登录 |
| /profile | 个人资料 |
| /job-expectation | 求职期望（行业偏好） |
| /applied-history | 投递记录 |
| /search | 搜索岗位 |
| /setting | 设置 |

## 行业偏好（求职期望）

平台提供 20+ 行业分类，用户可多选。用户已选：
- 資訊科技（子选项：業務/系統分析師、數據庫開發、技術顧問、產品管理&開發、軟件工程師、技術-項目管理）
- 市場推廣/傳媒
- 銷售
- 行政
- 製造/物流
- 工程師
- 科技（如是独立分类）

未勾（黑名单）：保險、地產

### 資訊科技子选项中没有直接 AI 选项
最接近的：業務/系統分析師、數據庫開發、技術顧問

## 推荐质量评估

算法匹配质量偏低，首屏推荐 2 个岗位都是保险/理财马甲：
1. 市場發展與客戶體驗專員（實为保险代理，技能标签含「人壽保險」）
2. 香港家族办公室合伙人（实为保险销售，"合伙人"=代理人）

**识别信号**：
- 「管理顧問/服務」+ 不限經驗 + 高傭金 → 保险马甲
- 「經營個人IP」+「體驗式行銷」+「引導式提問」→ 保险活动拉人
- 「無特定產業經驗要求」+「全方位支持你的發展」→ 来者不拒
- 「建立並管理團隊，晉升區域負責人」→ 金字塔拉人头
- 技能标签含「人壽保險」→ 直接暴露

- 不作为主力求职平台
