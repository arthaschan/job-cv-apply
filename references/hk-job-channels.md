# 香港求职渠道全景 + 新渠道搜索方法

> 来源: 2026-06-10 抖音信息差视频 + 实际调研
> 核心认知: JobsDB/LinkedIn 只覆盖香港招聘约30-40%。大量"双休、准点下班、对内地人友好"的中资/央企/外企后台岗走内部渠道。

---

## 渠道分层

### 第一层：已自动化（cron / API）

| 渠道 | 方法 | 频率 | 备注 |
|------|------|------|------|
| Gmail扫描 | gmail_job_scanner.py | 每日10:00 cron | LinkedIn+JobsDB邮件推送 |
| JobsDB搜索API | curl JSON | 按需 | keywords参数，pagesize小写 |

### 第二层：半自动（用户操作 + agent分析）

| 渠道 | 用户操作 | agent角色 | 频率建议 |
|------|---------|----------|---------|
| **简职HK（公众号）** | 看推送→截图/转发 | 匹配分析 | 每日 |
| **劳工处** www2.jobs.gov.hk | browser搜"管理/行政"类别 | 匹配分析 | 每周2-3次 |
| **CTgoodjobs** ctgoodjobs.hk | browser搜关键词 | 匹配分析+JD核实 | 每周2-3次 |
| **猎聘** liepin.com | 登录看推荐 | 匹配分析 | 每周 |
| **Jijis** jijis.org | 注册+搜索 | 匹配分析 | 每周 |

### 第三层：手动（需用户自行操作）

| 渠道 | 说明 |
|------|------|
| 中资/央企官网 | SPA网站，需逐个browser打开 |
| 行业WhatsApp/微信群 | 内推/急招，agent无法访问 |
| Glassdoor | 投递前查双休/加班评价 |
| 微信公众号（其他） | 各行业招聘号 |

## 劳工处搜索方法

**URL**: https://www2.jobs.gov.hk/0/cn/jobseeker/jobsearch/search/

**特点**:
- ASP.NET 表单网站（非SPA，browser可交互）
- **中文版必须用中文关键字**（英文关键字在中文版返回0结果）
- 类别"管理/行政"可筛掉蓝领/基础岗
- 结果含薪资范围+工作地点
- 搜索结果较少（管理岗通常个位数），但质量不低

**搜索建议**:
- 关键字: "经理 总监" 或 "管理 运营" 或 "总经理"
- 类别: "管理/行政"
- 薪资: 设置下限$20,000+过滤低薪岗
- 排序: "薪酬由多至少" 优先看高薪岗

**注意**: 持有效工作签证，可网上应征。

## CTgoodjobs搜索方法

**URL**: https://www.ctgoodjobs.hk/zh

**特点**:
- SPA网站，browser可交互
- 需接受cookies弹窗
- 搜索结果含公司名+岗位标题
- **需注册才能看完整JD**
- 薪资透明度高于JobsDB

**搜索建议**:
- 英文关键词效果更好（"operations director"、"business development"）
- 可按工种/地点/薪资筛选
- "月薪$25k+好工" 专区可直接浏览

## 简职HK

- **纯微信公众号**，无网页版，agent无法直接访问
- 80%+ 中资/央企/内地背景公司
- 特点: 超多双休、朝九晚六、不强制粤语
- 很多岗位只在这发，不在JobsDB公开
- **操作方式**: 用户每日查看推送，看到合适的截图/转发给agent分析

## 信息差核心原则

| 大众平台 | 小众/内部渠道 |
|---------|-------------|
| JobsDB/Indeed | 简职HK/CTgoodjobs/劳工处 |
| 岗多、杂、内卷大 | 岗精、好公司多 |
| 双休比例低 | 双休比例高 |
| 竞争大 | 竞争小 |

**结论**: 先补齐渠道，再海投。

---

## Pitfall

1. **劳工处中文版用中文关键字** — 英文关键字返回0结果
2. **CTgoodjobs需注册** — 不注册只能看岗位标题，看不到JD
3. **简职HK是微信封闭生态** — agent无法自动化，依赖用户转发
4. **中资官网全是SPA** — curl拿不到数据，需逐个browser交互
5. **不要只搜科技企业** — 中资传统行业（基建/贸易/物流）的管理岗更匹配"双休+对内地人友好"
6. **JobsDB API不识别中文公司名** — 搜中资必须用英文名（如"Orient Overseas"而非"东方海外"）
