# 求职申请表常见字段填写模板

## 标准字段速查

| 字段 | 英文填写 | 中文说明 |
|------|---------|---------|
| Earliest start date | `Immediately` | 持有效工作签证，随时入职 |
| Expected compensation (USD) | `$40,000 - $50,000 / year` | ≈HKD 26K-33K/月，远程初创合理价位 |
| Expected compensation (HKD) | `HKD XXX,XXX / year` | 根据实际情况填写 |
| University & study | `某211大学 — BEng Vehicle Engineering; 某高校 — MSc Applied Artificial Intelligence (2026)` | 某211大学+某高校 |
| Current location | `Hong Kong` or `Hong Kong SAR` | |
| Sponsorship required | **No** | 有效工作签证，不需sponsor |
| PM work experience | **Yes** | 产品定位+用户增长+门店运营=广义PM |
| High-intensity environment | **Yes** | 直接选Yes |
| LinkedIn Profile | `https://www.linkedin.com/in/your-linkedin-id-your-linkedin-id` | 备用编码版: `%E9%B8%BF%E7%92%8B-%E5%8D%A2-your-linkedin-id` |

## ⚠️ 薪资单位陷阱

| 问题措辞 | 正确填写 | 错误做法 |
|---------|---------|---------|
| Total Compensation / 总薪酬 | **年薪** (HKD/年) | ❌ 填月薪 |
| Monthly Salary / 月薪 | 月base (HKD/月) | ❌ 填年薪 |
| Expected Salary (不指定) | 视上下文，通常年薪 | — |
| USD (远程岗位) | $40K-50K/年 (远程初创) | ❌ 用HK标准填 |

> 如只填一个数字：HKD年薪填900K，USD年薪填45K。
> 备注栏加: *Flexible and open to discussion based on the full compensation package.*

## ⚠️ US/EST时区问题

用户在UTC+8(香港)，EST=UTC-5，差13小时。EST工作时间=北京时间凌晨0-8点。

- 有年幼子女 → 凌晨工作不现实
- **建议选No**，诚实比拿到offer后拒好
- 如果对方因此拒了，说明岗位本身不合适

## ⚠️ 粤语要求判断

- "Bilingual Cantonese + English" = 硬性要求 → ❌ 基础不够
- "Cantonese preferred" / "Cantonese a plus" = 加分项 → ⚠️ 可投
- 不提粤语 = ✅ 无要求
